## 快速开始
### 服务器端

```sh
git clone git@github.com:WeneShan/TCPclient.git
cd TCPclient
uv venv
source .venv/bin/activate
cd server 
python server.py
```
### 客户端
```sh
git clone git@github.com:WeneShan/TCPclient.git
cd TCPclient
uv venv
source .venv/bin/activate
python client.py
```

### 测试
```sh
git clone git@github.com:WeneShan/TCPclient.git
cd TCPclient
uv venv
source .venv/bin/activate
uv pip install -r requirements.txt
# 进入测试目录
cd test/
 
```
